Taxonomy

The OSS Taxonomy helps to classify a Free and Open-Source (FOSS) project. It is important because classification of a project shows the relationships between it and its contemporaries. It also helps the user to compare between similar projects and choose one that best suits their need. Classification of project is one of the prerequisites before it can be compared and ranked in a meaningful way.

To achieve this goal, OSSRank has defined an initial set of categories and associated a set of keywords with them. These categories ad keywords are continously updated as new generation technologies or project types appear.
